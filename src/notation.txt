Это краткое руководство к пользованию и настройке сайта:
/pages - страницы сайта;
/components - функциональные и классовые компоненты которые используются для рендеринга страниц;
/sql - функции для взаимодействия с базой данных;
/store - хранилище информации, с использованием mobx;
-------------------------------------------------------------------------------------------
/pages/AboutPage - страница about us;
/pages/AddPage - страница add, которая появляется если зайти на сайт как админ;
/pages/AuthPage - страница входа;
/pages/DrawerPage - страница корзины;
/pages/MainPage - главная страница;
/pages/ProfilePage - страница профиля;
/pages/CatalogBrandPage - каталог кроссовок определенного бренда;
/pages/CatalogPage - каталог всех кроссовок;
/pages/SneakerDetailPage - страница определенной пары кроссовок;
-------------------------------------------------------------------------------------------
/components/addBrand - форма добавление бренда на addPage;
/components/addSneaker - форма добавление пары кроссовок на addPage;
/components/brandCard - карточка бренда;
/components/brandList - список брендов на MainPage;
/components/error - вывод ошибок при регистрации или входе на сайте;
/components/loginForm - форма входа на сайт на AuthPage;
/components/signinForm -форма регистрации на сайте на AuthPage;
/components/slider - верстка автоматического слайдера;
/components/sneaker - верстка детальной страницы выбранной пары кроссовок;
/components/sneakerBrandList - список кроссовок выбранного бренда;
/components/sneakerCard - карточка пары кроссовок для каталога;
/components/sneakerList - список кроссовок;
/components/sneakerDrawerCard - карточка пары кроссовок для корзины;
/components/sneakerDrawerList - список кроссовок для корзины;
-------------------------------------------------------------------------------------------
/sql/getSneakers - получение все кроссовок;
/sql/getBrands - получение всех брендов;
/sql/getUsers - получение всех пользователей;
/sql/addSneaker - добавление модели кроссовок;
/sql/addBrand - добавление бренда кроссовок;
/sql/addUser - добавление пользователя;
/sql/mysqlConnect - установка соедидения с базой данных;
/sql/mysqlClose - завершение соединения с базой данных;
-------------------------------------------------------------------------------------------
/store/logStore - считывание и хранение информации о пользователе в течении одного сеанса, до перезагрузки страницы;
/store/sneakerStore - считывание и хранение добавленных в корзину моделей кроссовок в течении одного сеанса, до перезагрузки страницы;
-------------------------------------------------------------------------------------------
public/brands_logo - логотипы брендов
public/img_adv - рекламные картинки
public/img_sneaker - фотографии каждой модели кроссовок, по 3 на каждую модель
public/person - картинка пользователя по умолчанию
public/logo - логотип магазина
